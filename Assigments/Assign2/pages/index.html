My To-Do List

Currently my routes allow for several ways to manipulate the to-do list data. You can view all tasks with "/api/todo", view only open tasks with "/api/todo/open", view completed tasks with "/api/todo/completed", view a specific task with "/api/todo/:id", create new entries with "/api/todo/", edit entries with "/api/todo/:id", and delete entries with "api/todo/:id"
